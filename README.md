# Day 11 — Unit Converter

**120 Days of Code** | NIT Rourkela | Mechanical Engineering

## What this is
A multi-category unit converter built with HTML, CSS and JavaScript.
Converts length, weight, temperature and speed.

## How to view
Open `index.html` in your browser.

## Features
- 4 categories: Length, Weight, Temperature, Speed
- Shows all conversions at once
- Swap units button
- Clean dark UI
